Propiedad padding

La propiedad CSS padding se utiliza para definir el espacio entre el contenido de un elemento y su borde. Esta, acepta uno, dos, tres o cuatro valores, que definen el espacio en los cuatro lados del elemento (superior, derecho, inferior e izquierdo) en ese orden. 

                
/* Aplicando un valor único a todos los lados */
div {
    padding: 10px;
}

/* Aplicando dos valores para arriba/abajo y derecha/izquierda */
p {
    padding: 15px 10px;
}

/* Aplicando tres valores para arriba, derecha/izquierda y abajo */
ul {
    padding: 10px 5px 20px;
}

/* Aplicando valores para todos los lados */
a {
    padding: 5px 10px 15px 20px;
}

Propiedad margin

La propiedad CSS margin se utiliza para definir el espacio entre los bordes exteriores de un elemento y los elementos circundantes (otros elementos HTML, como divs, párrafos, imágenes, etc.). En esencia, el margen controla la separación entre un elemento y los elementos que lo rodean.
La propiedad margin también acepta uno, dos, tres o cuatro valores, y su funcionamiento es similar al de la propiedad padding, pero en lugar de crear espacio alrededor del contenido de un elemento, crea espacio alrededor del elemento en sí. 

                
/* Aplicando un valor único a todos los lados */
div {
    margin: 20px;
}

/* Aplicando dos valores para arriba/abajo y derecha/izquierda */
p {
    margin: 10px 30px;
}

/* Aplicando tres valores para arriba, derecha/izquierda y abajo */
ul {
    margin: 5px 15px 10px;
}

/* Aplicando valores para todos los lados */
a {
    margin: 5px 10px 15px 20px;
}

Propiedad border

La propiedad CSS border se utiliza para definir los bordes alrededor de los elementos HTML. Estos son útiles para darle un aspecto más visual a los elementos, separarlos de otros elementos o para resaltarlos de alguna manera. La propiedad border tiene varias subpropiedades que se utilizan para personalizar los bordes. Estas subpropiedades son:

  • border-width: Define el ancho del borde. Puedes establecer valores en píxeles, em, rem o porcentajes.
  • border-style: Define el estilo del borde, como sólido, punteado, discontinuo, doble, entre otros.
  • border-color: Establece el color del borde. Puede ser un valor de color en formato hexadecimal, nombre del color o RGB. 

                
elemento {
    border: 2px solid #0077b6;
}
